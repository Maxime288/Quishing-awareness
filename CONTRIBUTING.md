# 🤝 Guide de contribution

Merci de vouloir contribuer à ce projet de sensibilisation au quishing !

## Comment contribuer ?

### 1. Signaler une erreur ou suggérer une amélioration
Ouvrez une **Issue** GitHub en décrivant :
- Ce qui est incorrect ou incomplet
- Votre suggestion d'amélioration
- Des sources si possible

### 2. Proposer du contenu
Vous pouvez contribuer :
- 📝 De nouveaux exemples d'attaques documentées
- 🖼️ Des visuels ou infographies (sous licence libre)
- 🌍 Des traductions dans d'autres langues
- 🧠 De nouvelles questions de quiz

### 3. Processus Pull Request
1. Forkez le dépôt
2. Créez une branche descriptive : `feature/exemple-quishing-sante` ou `fix/correction-quiz-q3`
3. Faites vos modifications
4. Vérifiez que le Markdown est bien formaté
5. Soumettez une Pull Request avec une description claire

## Code de conduite
- Contenu éducatif uniquement — aucune technique d'attaque réelle
- Sources citées obligatoires pour les faits
- Respectueux et bienveillant dans les échanges

## Merci à tous les contributeurs ! 🙏
